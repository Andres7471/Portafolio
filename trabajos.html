Trabajos

Peaje Simulator

Este proyecto tuvo como objetivo explicar, a través de un juego, la implementación de sistemas de escaneo de placas vehiculares mediante la tecnología Flypass, con el fin de fortalecer la seguridad.

SuperMarket

Este proyecto implementó, utilizando Java, un sistema de supermercado virtual con las siguientes funcionalidades: gestión de inventario, compra de productos en línea y gestión de usuarios con roles diferenciados mediante inicio de sesión.

Travel Expenses

Se está desarrollando una página web para el registro y control de gastos incurridos en el exterior, con el fin de optimizar la planificación de futuros gastos. La plataforma incorporará modelos de inteligencia artificial para facilitar la asignación de precios a empleados, y se mantendrá un registro completo de transacciones para evaluar la justificación de los gastos.
